## ‚ú® Key Features

### üè¢ Intelligence Hub (Admin Panel)
Manage your organization's proprietary data blocks.
- **Client Vault:** A secure "folder" containing all user identities and registration metadata.
- **Storage Blocks:** Add, edit, and index text documents for RAG processing.
- **Strict Access Control:** Administrator rights are exclusively locked to authorized personnel (e.g., `emanabdulsemed4398@gmail.com`).

### ü§ñ Gemini 3 Powered Chat
A sophisticated floating widget that leverages the latest LLM capabilities.
- **RAG-Grounded Answers:** Responses are strictly limited to the data provided in the Intelligence Hub to ensure zero-hallucination.
- **Streamlined UI:** Minimizable chat interface with smooth typing animations and message threading.

### üì° Production Monitoring (The Live Console)
A developer-first view of the system's "under the hood" operations.
- **Real-time Traffic:** Watch `POST`, `GET`, and `PUT` requests as they hit the API.
- **DB Explorer:** Inspect `TBL_USERS` and `TBL_KNOWLEDGE` directly from the UI.
- **System Metrics:** Live tracking of uptime, storage utilization, and mean latency.

## üö¶ Operational Flow

1. **Authentication:** Sign in via traditional email or choose an account via the **OAuth Account Picker** (Google/Facebook/GitHub).
2. **Ingestion:** Navigate to the **Intelligence Hub** and commit "Intelligence Blocks."
3. **Observation:** Monitor the **Live Console** on the right to see the database commit in real-time.
4. **Interaction:** Open the **Chat Widget** to ask questions. The bot will retrieve relevant blocks and provide a grounded response.
